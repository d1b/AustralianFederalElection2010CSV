Help

The download files above are provided to allow you to create your own database or spreadsheet to further analyse the election results.

  • The election result downloads have been provided in Microsoft Excel Comma-Separated-Value (CSV) format and Tab-Delimited-Value (TAB) format.
  • Both of these file formats are text based and can be opened with a simple text editor program like Microsoft Notepad.
  • The CSV format is most suited to working with spreadsheet applications such as Microsoft Excel.
  • The TAB format is most useful when you wish to import the election results into a database system.

Notes:

  1. The best way to download a file depends on your computer, browser and security settings. For example, if you have Microsoft Excel installed on your computer and you click on a link to a CSV download then in most cases the download will open automatically in Microsoft Excel for you. However if this is not the case most browsers provide a feature that will let you download a file. Once you have downloaded the file and saved it to your system you can analyse the election results using the software of your choice.
  2. These results are not final.
